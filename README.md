# Web Terminel Chat

Web chat room, like a terminal

![](https://ci.cncn3.cn/49deb6d1b30fdfd539bece55d250dc4b.png)

------

### Backend

```
cd backend
npm i
node main.js
```

### frontend

1. Change `src/config.json`

```json
{
    "server": "the server ws url"
}
```

2. build

```
npm i
npm run build
```
